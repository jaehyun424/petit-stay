## 1. 제품 정의

**Petit Stay는 한국을 여행하는 외국인 가족이 숙소 유형에 관계없이 검증된 다국어 시터를 직접 검색·선택·예약·결제할 수 있는 베이비시팅 매칭 플랫폼이다.**

- 거래 구조: **C2C 매칭** — 부모가 시터를 직접 검색·선택·예약·결제
- 채널 구조: 호텔·레지던스·호스트·OTA는 **referral / distribution partner**
- 시터 관계: 직접고용 아님. **비전속 파트너 또는 독립 서비스 제공자**. 가격/가용시간을 스스로 설정하고, 건별 수락/거절
- 수익: 거래 수수료 **15~20%** (호텔 프리미엄 25~30%)

## 2. V2가 이전(V1)과 다른 점

| V1 (이전) | V2 (현재) |
|-----------|-----------|
| B2B2C — 호텔이 예약 생성, 플랫폼이 시터 배정 | C2C 매칭 — 부모가 직접 시터 검색·선택 |
| 호텔 전용 서비스 | 숙소 무관 (호텔·에어비앤비·민박·게스트하우스) |
| Gold/Silver 등급 고정 요금 | 시터 자율 가격 + 플랫폼 권장 밴드 |
| Hotel Console = 운영 대시보드 | Partner Console = referral 현황 + QR 발급 |
| Trust Protocol 4단계 | Trust Stack 5층 |
| 리뷰 없음 | Verified Review 시스템 (핵심 기능) |

## 3. 사용자군

### 부모 (핵심 사용자 #1)
- 한국 여행 중 외국인 가족 (미국·일본·중국·대만·홍콩 등)
- 3~8세 자녀 동반, 저녁/야간 돌봄 필요
- **핵심 니즈**: 낯선 나라에서 3분 안에 검증된 시터를 찾고 예약
- **언어**: 영어·일본어 우선, 중국어(번체→간체) 단계적

### 시터 (핵심 사용자 #2)
- 서울 거주 다국어 가능 돌봄 경험자
- **핵심 니즈**: 프리미엄 외국인 고객, 높은 시급(₩15,000~50,000), 유연한 일정
- **프로필**: 사진, 30~60초 자기소개 영상, 자격증, 경력, 언어 등급, Certified 배지, 리뷰, 가격, 가용시간

### 파트너 (채널)
- 호텔 컨시어지, 서비스드 레지던스, 에어비앤비 호스트
- **역할**: QR/링크/가이드북으로 투숙객에게 Petit Stay 안내
- **호텔은 고용주가 아님**. 추천 채널(referral partner)

### 관리자 (내부)
- 운영, 시터 검증, 분쟁 처리, 정산
- **비공개** — 런칭 제품에서 보이지 않음

## 4. 핵심 플로우 — Smith 가족 스토리보드

1. 부모가 "babysitter Seoul" 구글 검색 → Petit Stay 발견
2. 날짜·시간·아이 나이·위치 입력 → 시터 5명 비교
3. 프로필(사진·영상·자격·리뷰·시급) 확인 → 시터 선택
4. 아이 정보(알레르기·루틴) + 비상연락처 입력 → 동의 → 결제(Visa, 에스크로)
5. 시터 수락 → 도착 → 디지털 체크인
6. 세션 수행(놀이·간식·수면 루틴) → 종료 리포트 자동 발송
7. 부모 리뷰 → 에스크로 정산(수수료 차감)

**이 플로우가 V2의 메인 데모이자 핵심 제품.**

## 5. V1 서비스 범위

| 포함 | 제외 |
|------|------|
| 서울 | 부산/제주/해외 |
| 18:00~23:00, 24시간 전 예약 우선 | overnight, 즉시 배차 |
| 3~8세, 형제 포함 최대 2명 | 영아, 고난도 특수케어 |
| in-room 또는 숙소 내 안전 공간 | 수영장, 차량 이동, 야외 |
| 놀이·식사 보조·수면 루틴·안전 관찰·부모 보고 | 투약, 의료, 목욕, 숙박 동반 |

## 6. Trust Stack 5층

1. **신원·경력·자격 확인** — 프로필이 아니라 검증 항목을 보여줌
2. **사전 기대치 정렬** — 서비스 범위, 금지 업무, 사진 정책, 비상 연락, 종료 기준
3. **현장 투명성** — 도착 확인, 인계, 종료, 세션 리포트
4. **이슈 대응** — 연락 우선순위, 호텔 통보, 기록, 후속 조치
5. **반복 사용** — 재예약 시 선호 정보 안전하게 재사용

## 7. Verified Review 시스템

- 완료된 유료 세션 후에만 리뷰 가능 (가짜 리뷰 방지)
- 부모→시터 공개 리뷰 + 시터→부모 내부 평가 (이중 구조)
- 표시 지표: 평점(5점), 후기 수, 응답률, 수락률, 취소율, 반복예약률
- 신뢰 배지: 언어 등급, CPR/응급처치, 신원확인 완료, 영상소개, 레퍼런스 확인

## 9. 디자인 원칙

- **"외국인 부모가 3분 안에 검증된 시터를 찾고 예약할 수 있는 다국어 매칭 플랫폼"**
- AI스러운 템플릿/SaaS 대시보드 UI 절대 금지
- 국내 20대 감도 앱(에이블리·29CM·지그재그) + sitter marketplace(Care.com·UrbanSitter·Sittercity) 혼합 벤치마크
- 분위기: 다정하고 세련되고 신뢰감 있게. 유치하지 않게.
- 핑크 토끼 모티프. 가볍기보다 따뜻하고 프리미엄.
- Quiet Luxury 톤 유지: Charcoal + Cream + Gold (+ 핑크 액센트)

## 11. 과장 금지 원칙 (GPT 팩트체크본 기반)

- "한국 최초/유일/완전 공백" ❌ → "소개형 시장은 존재하나 표준화 플랫폼은 부재" ⭕
- 확정 안 된 성과를 사실처럼 ❌
- 공식 사실 / 동적 스냅샷 / 내부 가정 / 사전 확인 필요 — 4단계 구분
- 법무 구조는 "방어 가능한 운영 가설"로. 최종 확정 표현 안 씀.
